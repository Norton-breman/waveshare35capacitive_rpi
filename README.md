# waveshare35capacitive_rpi

enable waveshare 3.5" capacitive touche screen (https://www.waveshare.com/wiki/3.5inch_Capacitive_Touch_LCD) to be used with raspberry pi.
Tested on rpi3B+ and zeros 2W
Tested OS:
  raspberry os lite bullseye
  dietpi trixie

Edit pin numbers corresponding to your wiring in install_waveshare35_ft6236.sh
sudo chmod +x install_waveshare35_ft6236.sh

sudo ./install_waveshare35_ft6236.sh
